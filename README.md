## 🔧 策略特点

### 入场条件
- **三连涨**: 连续3天收盘价不减 A≤B≤C
- **V型底**: 第0天是过去n天(n≥20)的最低点

### 风险管理
- **硬止损**: 3.1%固定止损
- **分级减仓**: 跌幅0-2%减仓10%，>2%减仓20%
- **豁免期**: 买入后第一天亏损不计入减仓

### 特征提取
- **价格动量**: 价格变化趋势
- **成交量动量**: 成交量变化趋势
- **波动率**: 日内价格波动
- **平均成交量**: 3日平均成交量
- **平均成交额**: 3日平均成交额
- **平均涨跌幅**: 3日平均涨跌幅

## 📈 优化目标

### 当前状态
- 四连涨策略收益率: 1.08%
- 三连涨策略收益率: 1.8%

### 优化方向
1. **特征组合**: 多维度特征提取和筛选
2. **参数优化**: 动态调整止损和减仓参数
3. **信号过滤**: 基于特征值预判信号质量
4. **组合策略**: 多策略组合提升稳定性
